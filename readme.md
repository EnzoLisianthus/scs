# 결혼 의사결정 로직 트리 (32세 / 데이터 분석 관리자)

## 1. 어떤 배우자와 결혼할 것인가
### 1.1 상대방 자체는 매력적인가
#### 1.1.1 외모
- 내 기준 호감인가
- 장기적으로 유지 가능한 스타일인가

#### 1.1.2 내면
- 성격 (배려, 안정성, 감정기복)
- 가치관 (돈, 일, 가족, 삶의 방향)
- 대화/소통 능력

#### 1.1.3 관계 궁합
- 갈등 해결 방식
- 함께 있을 때의 편안함
- 장기 관계 유지 가능성

---

### 1.2 상대방 외적인 요소는 어떤가
#### 1.2.1 직업/경제력
- 안정성
- 성장 가능성

#### 1.2.2 가족/환경
- 가족 분위기
- 결혼 후 영향도

#### 1.2.3 생활 방식
- 소비 습관
- 시간 사용 방식
- 미래 계획 (거주, 커리어)

---

### 1.3 나와의 현실적 적합성
- 내 커리어와 충돌 없는가
- 라이프스타일 일치
- 결혼 타이밍 적절성

---

## 2. 나는 결혼 준비가 되어 있는가
- 경제적 준비
- 정신적 안정성
- 책임 감당 가능 여부

---

## 3. 친구들의 결혼으로 인한 감정 (시기심)
### 3.1 감정의 원인
- 비교 심리
- 뒤처진 느낌
- 사회적 압박

### 3.2 감정의 위험성
- 잘못된 선택 유도 (조급한 결혼)
- 기준 하향 가능성

### 3.3 대응 전략
- 기준 유지 (조건 명확화)
- 속도보다 방향 중시
- 데이터 기반 판단 (감정 vs 사실 분리)

---

## 4. 최종 의사결정 기준
- "좋은 사람" vs "나에게 맞는 사람"
- 단기 감정 vs 장기 안정성
- 후회 최소화 기준 (5년 후 관점)
